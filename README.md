# Kinfolk Inspired Astro Blog Template

Minimalist, typography-focused blog template for writers and knowledge curators.

[Live Demo](https://emintham.com) | [Changelog](https://github.com/emintham/blog-template/blob/master/CHANGELOG.md)

## Features

- **Minimalist Design:** Clean, Kinfolk-inspired typography
- **Astro Powered:** Fast, modern, content-focused
- **Responsive & SEO Friendly:** Automatic sitemap generation
- **Optimized Images:** WebP conversion with responsive delivery

## Content Types

- **Standard Posts:** Long-form articles
- **Fleeting Thoughts:** Short-form insights, fully displayed in previews
- **Book Notes:** Reviews with book covers, collapsible quotes, and dedicated tagging

## Organization

- **Tags:** General post tags and book-specific tags
- **Quote Tags:** Tag individual quotes for cross-referencing
- **Series:** Group related posts chronologically
- **Sidenotes:** Tufte-style margin notes

## Admin Tools (Dev Mode)

- Live preview and hot reload
- Interactive tagging
- RAG-powered semantic search and AI writing assistant
